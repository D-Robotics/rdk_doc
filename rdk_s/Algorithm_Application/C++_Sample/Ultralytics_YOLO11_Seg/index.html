е®һдҫӢеҲҶеүІ-Ultralytics YOLO11

жң¬зӨәдҫӢеұ•зӨәдәҶеҰӮдҪ•еҹәеңЁ BPU дёҠиҝҗиЎҢ YOLOv11 иҜӯд№үеҲҶеүІжЁЎеһӢпјҢж”ҜжҢҒеӣҫеғҸйў„еӨ„зҗҶгҖҒжҺЁзҗҶгҖҒеҗҺеӨ„зҗҶпјҲи§Јжһҗиҫ“еҮәе№¶еҸ еҠ еҪ©иүІеҲҶеүІжҺ©з ҒпјүзӯүеҠҹиғҪпјҢжң¬зӨәдҫӢд»Јз ҒдҪҚдәҺ/app/cdev_demo/bpu/03_instance_segmentation_sample/02_ultralytics_yolo11_seg/зӣ®еҪ•дёӢгҖӮ

жЁЎеһӢиҜҙжҳҺвҖӢ

  • з®Җд»Ӣпјҡ

    Ultralytics YOLO11 жҳҜдёҖж¬ҫиҪ»йҮҸзә§зӣ®ж ҮжЈҖжөӢдёҺе®һдҫӢеҲҶеүІжЁЎеһӢпјҢеҹәдәҺ YOLO зі»еҲ—и®ҫи®Ўе№¶иһҚеҗҲдәҶ anchor-free дёҺ anchor-based жҖқжғіз»“жһ„дёҺеӣһеҪ’еҲҶз®ұпјҲdistributional regressionпјүзӯ–з•ҘгҖӮжң¬жЁЎеһӢдёәе…¶е®һдҫӢеҲҶеүІеҸҳдҪ“пјҢж”ҜжҢҒеҗҢж—¶иҫ“еҮәиҫ№з•ҢжЎҶгҖҒзұ»еҲ«жҰӮзҺҮе’Ңй«ҳиҙЁйҮҸзҡ„еғҸзҙ зә§жҺ©иҶңпјҢйҖӮз”ЁдәҺе®һж—¶еңәжҷҜдёӯзҡ„еӨҡеҜ№иұЎжЈҖжөӢдёҺеҲҶеүІд»»еҠЎгҖӮ

  • HBM жЁЎеһӢеҗҚз§°пјҡ yolo11n_seg_nashe_640x640_nv12.hbm

  • иҫ“е…Ҙж јејҸпјҡ NV12 ж јејҸеӣҫеғҸпјҲY/UV еҲҶзҰ»пјүпјҢе°әеҜёдёә 640x640

  • иҫ“еҮәпјҡ

    • зӣ®ж ҮжЈҖжөӢз»“жһңпјҲbounding box + зұ»еҲ« + еҲҶж•°пјү

    • е®һдҫӢеҲҶеүІжҺ©иҶңпјҲжҜҸдёӘзӣ®ж ҮеҜ№еә”дёҖеј  maskпјү

еҠҹиғҪиҜҙжҳҺвҖӢ

  • жЁЎеһӢеҠ иҪҪ

    еҠ иҪҪйҮҸеҢ–еҗҺзҡ„ Ultralytics YOLO11 е®һдҫӢеҲҶеүІжЁЎеһӢпјҢе№¶и§ЈжһҗиҝҗиЎҢж—¶е…ғж•°жҚ®гҖӮ

  • иҫ“е…Ҙйў„еӨ„зҗҶ

    е°Ҷиҫ“е…Ҙ BGR еӣҫеғҸзј©ж”ҫиҮі 640Г—640пјҢе№¶иҪ¬жҚўдёә NV12 ж јејҸпјҲYгҖҒUV е№ійқўеҲҶзҰ»пјүпјҢд»ҘйҖӮй…ҚжЁЎеһӢиҫ“е…ҘиҰҒжұӮгҖӮ

  • жҺЁзҗҶжү§иЎҢ

    йҖҡиҝҮ .infer() ж–№жі•и§ҰеҸ‘еүҚеҗ‘жҺЁзҗҶпјҢжҺЁзҗҶиҫ“еҮәеҢ…еҗ«еӨҡдёӘе°әеәҰзҡ„зұ»еҲ«еҲҶж•°гҖҒиҫ№з•ҢжЎҶеӣһеҪ’гҖҒжҺ©иҶңзі»ж•°д»ҘеҸҠе…ЁеұҖжҺ©иҶңеҺҹеһӢеј йҮҸгҖӮ

  • з»“жһңеҗҺеӨ„зҗҶ

    • дҪҝз”ЁйҳҲеҖјзӯӣйҖүжЈҖжөӢеҖҷйҖүпјҢе№¶и§Јз Ғиҫ№з•ҢжЎҶдёҺжҺ©иҶңзі»ж•°пјӣ

    • еҗҲе№¶еҗ„е°әеәҰиҫ“еҮәпјҢеә”з”Ё NMS зӯӣйҖүжңҖз»Ҳзӣ®ж Үпјӣ

    • еҲ©з”ЁжҺ©иҶңеҺҹеһӢдёҺзі»ж•°жҒўеӨҚеҮәжҜҸдёӘзӣ®ж Үзҡ„жҺ©иҶңпјӣ

    • жҺ©иҶңеҸҠиҫ№з•ҢжЎҶеқҮиў«зј©ж”ҫеӣһеҺҹе§ӢеӣҫеғҸе°әеҜёпјӣ

    • ж”ҜжҢҒеҸҜйҖүеҪўжҖҒеӯҰж“ҚдҪңпјҲејҖж“ҚдҪңпјүд»ҘдјҳеҢ–жҺ©иҶңиҫ№зјҳпјӣ

    • жңҖз»Ҳиҫ“еҮәдёәиҫ№жЎҶгҖҒзұ»еҲ«гҖҒеҲҶж•°дёҺеғҸзҙ зә§е®һдҫӢжҺ©иҶңгҖӮ
