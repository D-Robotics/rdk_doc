# 1.2.4.1 Flashing Instructions


:::warning Notes

- Do not plug or unplug any devices while powered, except for USB, HDMI, and Ethernet cables.
- The RDK X5 Module is powered via the USB Type-C port on the carrier board. A power adapter capable of delivering 5V/5A must be used to power the development board.
- Do not power the development board from a computer's USB port, as insufficient power supply may cause abnormal behaviors such as **unexpected power-offs or repeated reboots**.

:::


## Boot Modes

The RDK X5 Module supports booting the system from either eMMC or an SD card.


### Booting from eMMC

#### Rufus Tool
    - Supports using local images
    - Supports Windows OS

### Booting from SD Card

Use either the RDK Studio tool or the Rufus tool.

#### RDK Studio Tool
    - Supports using local images and downloading images online
    - Supports Windows, Linux, and macOS
    - Supports SD card flashing and on-board SD card flashing

#### Rufus Tool
    - Supports using local images
    - Supports Windows OS
    - Supports SD card flashing and on-board SD card flashing
