# 1.2.4.3 Using the RDK Studio Tool


## SD Card Flashing

The RDK Studio tool provides system flashing functionality and enables device connection and management. It supports Windows, Linux, and macOS platforms. For detailed steps, refer to [Flashing the System Using RDK Studio](../../09_RDK_Studio/04_flashing.md).


## On-Board Flashing

### On-Board SD Card

1. Insert the SD card into the development board, connect the USB Type-C cable to your PC, press and hold the Sleep button (Pin 23), power on the development board, wait for 5 seconds, and the board will enter flashing mode. The SD card will then appear as a USB drive on your PC.
2. Follow the instructions in [Flashing the System Using RDK Studio](../../09_RDK_Studio/04_flashing.md) to complete the system flashing.


### eMMC Flashing

1. Connect the USB Type-C cable to your PC, press and hold the Sleep button (Pin 23), power on the development board, and wait for 5 seconds. The board will enter flashing mode, and the built-in eMMC on the core board will be mapped as a USB drive.

2. Follow the instructions in [Flashing the System Using RDK Studio](../../09_RDK_Studio/04_flashing.md) to complete the system flashing.
