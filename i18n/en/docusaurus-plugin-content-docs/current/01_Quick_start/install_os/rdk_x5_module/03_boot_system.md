# 1.2.4.3 Booting the system

## Default

By default the system boots from **eMMC**.

## Boot order

Use `srpi-config` to change boot order; see [srpi-config — Advanced Options](../../../02_System_configuration/02_srpi-config.md#advanced-options) and the **Boot Order** entry.
