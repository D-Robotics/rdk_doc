## **Frequently Asked Questions**

Common issues encountered when using the development board for the first time are as follows:

- **<font color='Blue'>Device does not power on</font>**:
  - Please ensure you are using the recommended power adapter.
  - Please ensure that the Ubuntu system image has been flashed onto the development board's Micro SD card or eMMC.
- **<font color='Blue'>Hot-plugging storage cards during operation</font>**: The development board does not support hot-plugging of Micro SD cards. If this occurs accidentally, please reboot the development board.
