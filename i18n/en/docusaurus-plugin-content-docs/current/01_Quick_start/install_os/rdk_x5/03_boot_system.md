# 1.2.3.3 Booting the system

Power off, insert the prepared microSD, connect HDMI to a monitor, then power on.

First boot runs default setup for about **45 seconds**; then the Ubuntu desktop should appear.

:::tip Board LED

**Green** on: system is booting; off or blinking means boot has finished.

:::

:::warning Note

If there is no display for more than **2 minutes**, use serial debug to check whether the board is running.

:::
