# 1.2.5.1 Flashing Instructions


:::warning Notes

- Do not plug or unplug any devices while powered, except for USB, HDMI, and Ethernet cables.
- The RDK Ultra development board is powered via a DC jack. It is recommended to use the power adapter included in the official kit or a power adapter rated at least **12V/5A**.
- Do not power the development board via a computer's USB port, as insufficient power supply may cause abnormal power-offs, repeated reboots, or other unexpected issues.

:::


## Flashing Modes

The RDK Ultra supports booting the system from eMMC mode.


## Flashing Tools

The RDK Ultra can use the PC-based tool hbupdate to flash the Ubuntu system, supporting both Windows and Linux platforms.
