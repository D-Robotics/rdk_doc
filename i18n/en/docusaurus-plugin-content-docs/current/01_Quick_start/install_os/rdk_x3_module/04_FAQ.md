# 1.2.2.4 FAQ

:::tip

For more answers, see [FAQ — hardware and system](../../../08_FAQ/01_hardware_and_system.md) and the [D-Robotics developer forum](https://developer.d-robotics.cc/forum).

:::

Common first-time issues:

- **<font color='Blue'>No boot after power-on</font>**:

  - Use a recommended power adapter.
  - Ensure Ubuntu is flashed to microSD or eMMC.

- **<font color='Blue'>USB host port not working</font>**: Disconnect any cable from the Micro USB data port.

- **<font color='Blue'>Hot-plugging the SD card</font>**: Not supported; reboot if the card was removed while running.
