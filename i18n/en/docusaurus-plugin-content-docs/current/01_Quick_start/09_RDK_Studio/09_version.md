# 1.9.9 Version Release

## RDK Studio-v0.3.18

### New Features

1. Added compatibility for using RDK S100 on RDK Studio

2. Added the latest images for RDK X3/X5

3. Added flashing support for RDK X3 Module, RDK X5 Module, and RDK S100

4. Added the Performance Node application for application performance monitoring
   
   ![Download Interface](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/02_System_configuration/image/cpu_frequency/rdk_studio/en/performance-node-install.png)

5. Added MacOS support for flash connection method

6. Added Wi-Fi pre-configuration after flashing


### Key Fixes

1. Fixed the issue where the shutdown and restart buttons were not functioning

### Optimizations

1. Display of empty storage devices during flashing

2. Initial language display for flashing module
