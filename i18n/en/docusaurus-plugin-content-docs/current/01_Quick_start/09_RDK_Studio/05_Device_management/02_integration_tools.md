## Using the Terminal

Click the terminal icon, and the Windows terminal will automatically pop up. Enter the user password to connect automatically.

:::info Note

The default password is the same as the login username:
- root-root
- sunrise-sunrise
  
:::
  
![RDK_Studio Terminal Interface](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_device_manage_hr_add_device_applicationspace_terminal.png)

## Using VS Code

Click the VS Code icon to automatically invoke the local VS Code Remote plugin for connection.

:::info Note

VS Code must be installed locally.

## Using Node-RED{#node-red}

1. Click the application space icon to view more applications.
   
   ![Application Space Interface](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_device_manage_hr_add_device_application_space_download.png)
   
2. Click to install Node-RED to run examples from the `Examples`.
   
   ![Download Interface](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_device_manage_hr_add_device_application_nodered_install.png)

3. Features like Jupyter that require installation can be installed as needed.

    ![RDK_Studio](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_device_manage_hr_add_device_applicationspace.png)

4. Click on a blank area to exit the application space.
