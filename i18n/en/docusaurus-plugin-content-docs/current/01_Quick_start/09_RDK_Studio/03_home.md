## Interface Introduction

1. When entering RDK Studio without an RDK device connected, a guided tour window will automatically pop up to help users understand the RDK device and its usage precautions. Click `Next` to view subsequent content, or click `Previous` to review earlier prompts.
     
    ![Guided Tour Window Interface Display](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_introduction2.png)

2. After reviewing all guided content, click `Finish` to close the pop-up window. You can also click the ` × ` in the top right corner to close the window at any time.    

    ![End Tour Interface Display](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_introduction3.png)  


## Home Page

After closing the guided tour pop-up, you will enter the RDK Studio home page:

![RDK Studio Home Page Interface Display](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_menu.png)

### Functional Area

- `Resources`: Here, you can add devices for management by clicking the `+Hardware` button in the top right corner.
- `Examples`: Here, you can directly install simple demos onto your development board.
- `Community`: This provides direct access to the D-Robotics Community, eliminating the need to open a web browser for reference.
- `NodeHub`: This provides direct access to NodeHub, offering a wealth of example nodes.
- `Imager`: This provides system flashing functionality via both card reader and flash connection methods.

### Language Settings and Reference Information

#### Language Settings

Click the language switch icon in the bottom left corner to expand language options and configure the interface language.

![Language Switch Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_language.png)

#### Learn More

Hover the mouse over the icons below the language settings to learn more about using RDK Studio.

![Usage Help Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction.png)

- Video Tutorials

    Click to select `Video tutorials` : The browser will jump to the Bilibili webpage. Click the title to watch the corresponding video guide.

    ![Video Guide Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction_video.png)

- Document

    Click to select the `Document` : Open the community in your browser, where you can browse forum topics related to RDK Studio.
    
    ![Community Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction_community.png)

- Forum

    Click to select the `Forum` : Open the community in your browser, where you can browse forum topics related to RDK Studio.
    
    ![Community Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction_community.png)

- Help Center

    Click to select the `Help Center` : Open the community in your browser, where you can browse forum topics related to RDK Studio.
    
    ![Community Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction_community.png)
