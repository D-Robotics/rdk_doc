# Samples Overview

## Introduction

This project provides multiple Python sample programs that use the BPU on the **RDK X5** platform, covering image classification, object detection, semantic segmentation, pose estimation, and other common tasks. The samples run inference with `.bin` quantized models so developers can quickly validate models and build applications.

On-board code path: `/app/pydev_demo/`
