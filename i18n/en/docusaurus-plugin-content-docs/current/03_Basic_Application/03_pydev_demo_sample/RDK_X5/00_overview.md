# Example Overview

## Example Introduction

This project includes multiple example programs for BPU usage written in Python, suitable for the RDK X5 (Software Version 3.5.0) platform, covering common tasks such as Image Classification, Object Detection, Instance Segmentation, and Pose Estimation. The examples use quantized models in.bin format for inference, facilitating developers to quickly verify model effectiveness and initiate application development. 

Board-side code location for this project: /app/pydev_demo/
