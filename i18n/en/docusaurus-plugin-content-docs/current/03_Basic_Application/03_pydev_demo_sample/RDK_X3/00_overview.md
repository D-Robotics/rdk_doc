# Samples Overview

## Introduction

This project provides multiple Python sample programs that use the BPU on **RDK X5** (software versions before 3.5.0) and **RDK X3**, covering image classification, object detection, instance segmentation, pose estimation, and other common tasks. The samples run inference with `.bin` quantized models so developers can quickly validate models and build applications.

On-board code path: `/app/pydev_demo/`
