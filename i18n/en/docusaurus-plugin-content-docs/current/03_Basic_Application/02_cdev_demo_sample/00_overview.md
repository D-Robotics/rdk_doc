# Example Overview

## Example Introduction

This project includes multiple methods written in C++ that use BPU, as well as multimedia-related sample programs, suitable for the RDK X3/X5 platform, covering examples of model inference, encoding and decoding, publish streaming, obtaining camera video streams, display, and serial pipeline examples of these modules. 
The board-side code location of this project: /app/cdev_demo/
