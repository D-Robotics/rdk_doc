# 1.7.4 Flashing the System

:::info Note

- If your storage device has already been flashed with the system, you can skip this chapter and proceed directly to [Adding RDK Device](../09_RDK_Studio/05_Device_management/01_hardware_resource.md).
- You need to use the third-party tool XBurn to flash the system.
:::

1. Click to download the flashing tool.
   
   ![Tool Download Interface](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/s100/install_os_tools.png)
   
2. Click to download the system image.
   
   ![Image Download Interface](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/s100/os_image.png)
   
3. Click `Learn More` to enter the product interface.
   
   ![Product Interface](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/s100/more.png)
