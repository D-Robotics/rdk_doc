# Opening Examples in Node-RED

```mdx-code-block
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
```

<Tabs groupId="rdk-type">
<TabItem value="windows" label="Windows">

1. Click `Examples` to enter the example list interface. Click the dropdown list at the bottom of an example application. All connected RDK devices will be displayed here. Select the RDK device on which you want to install the example application.
   
   ![RDK Device List Page](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_example_nodered_static_image_select_device.png)

2. Check if the example is available. An available example shows a green indicator dot in the upper left corner, while an unavailable one shows red.
   ![RDK Device List Page](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_example_application_use_status_light.png)

   :::info Note
   - Some examples are already installed. For uninstalled examples, click `Install Example` to install them. This section uses the `Gesture Recognition` example for explanation.
   - The Gesture Recognition example requires tracking gestures, so please connect a camera in advance.
   - If Node-RED is not installed, a prompt will appear. You need to install Node-RED in "Device Management - Application Space". Refer to [Using Node-RED](../04_Device_management/02_integration_tools.md#using-node-rednode-red).
   :::
   
3. Click Node-RED under the `Gesture Recognition` example. After reading the guide information, click `Close` to close the guide pop-up window.
   
   ![Guide Pop-up Page](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_example_application_gesture_help.png)

4. Enter the example application workflow interface.
        
   ![Gesture Recognition Example Page](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_example_nodered_gesture_detection.png)

</TabItem>
