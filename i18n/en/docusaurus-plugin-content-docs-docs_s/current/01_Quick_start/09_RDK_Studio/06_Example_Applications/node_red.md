# 1.7.6.1 Opening Examples in Node-RED

```mdx-code-block
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
```

<Tabs groupId="rdk-type">
<TabItem value="windows" label="Windows">

1. Click `Sample Applications` to enter the example list interface. Click the dropdown list at the bottom of the example application. All connected RDK devices will be displayed here. Select the RDK device on which you want to install the example application.
   
   ![RDK Device List Page](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_example_nodered_static_image_select_device.png)

2. Check if the example is available. When available, a green indicator dot is displayed in the upper left corner; when unavailable, it shows red.

    ![Example Application Status Light](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_example_application_use_status_light.png)

   :::info Note
   - Some examples are already installed. For uninstalled examples, click `Install Example` to install them.
   - For examples that require a camera, please connect the camera in advance. Both USB and MIPI cameras are supported. This chapter uses a USB camera as an example. The method to connect a USB camera is as follows:
  
       ![Connect Camera Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/connect_camera.png)

   - If Node-RED is not installed, a prompt will appear. You need to install Node-RED in "Device Management - Application Space". Refer to [Using Node-RED](../04_Device_management/02_integration_tools.md#using-node-red).
   :::
   
3. Click on Node-RED under the examples. After reading the tour information, click `Close` to close the tour popup window.
   
    ![Tour Popup Window](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_example_application_gesture_help.png)

4. Enter the example application workflow interface.
        
   ![Gesture Recognition Example Page](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_left_menu_example_nodered_gesture_detection.png)

</TabItem>
