import Tabs from '@theme/Tabs';

<Tabs groupId="rdk-type">
<TabItem value="windows" label="Windows">

## Language Settings

Click the language switch icon in the bottom left corner to expand language options and configure the interface language.

![Language Switch Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_language.png)

## Learn More

Hover the mouse over the icons below the language settings to learn more about using RDK Studio.

![Usage Help Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction.png)

### Video Tutorials

Click to select `Video tutorials` : The browser will jump to the Bilibili webpage. Click the title to watch the corresponding video guide.

![Video Guide Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction_video.png)

### Document

Click to select the `Document` : Open the community in your browser, where you can browse forum topics related to RDK Studio.

![Community Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction_community.png)

### Forum

Click to select the `Forum` : Open the community in your browser, where you can browse forum topics related to RDK Studio.

![Community Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction_community.png)

### Help Center

Click to select the `Help Center` : Open the community in your browser, where you can browse forum topics related to RDK Studio.

![Community Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_direction_community.png)

## Copyright Information

Hover the mouse over the copyright icon to view the complete copyright information.

![Copyright Information Image](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/en/rdk_studio_bottom_menu_copyright.png)

## Version Information
