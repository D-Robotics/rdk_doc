# 7.1.2 Accessory List

The Certified Accessory List refers to a list of third-party accessories officially certified by D-Robotics as compatible with the RDK S100 development board. It primarily includes categories such as basic accessories (power supplies, enclosures, heatsinks), cameras, displays, etc. The list will be updated periodically to gradually include more accessory models.

:::info

- The accessory list is updated regularly based on compatibility status.

:::
