## 终端使用

点击终端按钮，会自动弹起 Windows 终端，输入用户密码即可自动连接。
  
![RDK_Studio终端界面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/device_manage/rdk_studio_left_menu_device_manage_hr_add_device_applicationspace_terminal.png)

## Vscode使用

点击Vscode图标，即可自动调用本地Vscode Remote插件进行连接。
:::tip

使用 VS Code 的 SSH Remote 插件打开 RDK 设备中的实例工程，因此本地需安装 Vsocode 。

:::
​
![RDK_Studio VSCode界面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/device_manage/rdk_studio_left_menu_device_manage_hr_add_device_applicationspace_vscode.png)

## 其他功能

其他 Jupyter 等需要安装功能，可以按需安装。
