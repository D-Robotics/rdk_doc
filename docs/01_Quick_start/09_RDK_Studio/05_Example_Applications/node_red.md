# Node-RED 打开示例

```mdx-code-block
import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
```


<Tabs groupId="rdk-type">
<TabItem value="windows" label="Windows">

1. 点击 `示例应用` 进入示例列表界面，点击示例应用底部的下拉列表，已连接的所有 RDK 设备都会在此处显示，选择要安装示例应用的 RDK 设备。
   
   ![RDK 设备列表页面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/rdk_studio_left_menu_example_nodered_static_image_select_device.png)

2. 查看示例是否可用，可用时左上角显示绿色指示点，不可用时显示红色。

    ![RDK 设备列表页面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/rdk_studio_left_menu_example_application_use_status_light.png)

   :::info 提示
   - 部分示例已经安装，未安装示例可点击 `Install Example` 进行安装，本章节以 `手势识别` 示例为例进行说明。
   - 手势识别示例需跟踪手势，请提前连接好摄像头。
   -  如未安装 Node-RED 会弹出提示，需在 “设备管理 - 应用空间” 中安装 Node-RED，参考 [Node-RED使用](../04_Device_management/02_integration_tools.md#node-red-使用node-red)。
   :::
   
3. 点击 `手势识别` 示例下的 Node-RED，阅读完导览信息后点击 `关闭` ，关闭导览弹窗。
   
    ![导览弹窗页面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/rdk_studio_left_menu_example_application_gesture_help.png)

4. 进入示例应用流程界面。
        
   ![手势识别示例页面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/rdk_studio_left_menu_example_nodered_gesture_detection.png)

</TabItem>
