3.2.5 vio2display 示例介绍

示例简介​

vio2display 是一个位于 /app/cdev_demo 目录中的 C 语言接口 开发代码示例，用于演示如何使用 c 语言打开摄像头，并将摄像头采集的数据呈现在显示器上。参考这个示例，用户可以理解并开发相关应用。

效果展示​

可以看到，执行示例时，接到 RDK HDMI 的显示屏拍摄到了显示屏上的执行记录。

硬件准备​

硬件连接​

该示例不需要鼠标和键盘，所以这里连接了摄像头 HDMI 显示屏，网线接口，电源线

快速开始​

代码以及板端位置​

进入到 /app/cdev_demo/vio2display 位置，可以看到 vio2display 示例里面包含了 2 个文件

root@ubuntu:/app/cdev_demo/vio2display# tree
.
├── Makefile
└── vio2display.c

编译以及运行​

我们直接在该目录下使用 make 即可编译出 vio2display 可执行文件。

root@ubuntu:/app/cdev_demo/vio2display# tree
.
├── Makefile
├── vio2display
├── vio2display.c
└── vio2display.o

执行效果​

我们首先要关闭 light 服务 , 然后再执行 vio2display ，这里我们要注意摄像头的分辨率，根据摄像头的分辨率填写 -h 和 -w 参数。

详细是介绍​

示例程序参数选项说明​

我们可以直接执行目标文件来确认参数选项说明

root@ubuntu:/app/cdev_demo/vio2display# ./vio2display 
Usage: vio2display [OPTION...]
vio(sensors) display sample -- An example of display sensor's image

  -h, --height=height        sensor output height
  -w, --width=width          sensor output width
  -?, --help                 Give this help list
      --usage                Give a short usage message

Mandatory or optional arguments to long options are also mandatory or optional
for any corresponding short options.

其中
-h 是必填选项，代表输入视频的像素高度
-w 是必填选项，代表输入视频的像素宽度
-i 是必填选项，代表输入视频的文件路径

软件架构说明​

本示例主要是从摄像头捕获 YUV 和 RAW 图，逻辑相对简单，打开摄像头之后，调用 libspcdev 库提供的接口之后，可以直接获取到 YUV 和 RAW 图进行保存。
