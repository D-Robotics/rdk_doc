# RDK S100 开发工具包

## 版本：3.2.0

:::info 版本说明
该版本对应系统软件 **4.0.2-Beta** 版本，用户可在板端执行 `cat /etc/version` 命令确认系统软件版本号。
:::

## 工具包下载

### OE 开发工具包

**下载地址：**

