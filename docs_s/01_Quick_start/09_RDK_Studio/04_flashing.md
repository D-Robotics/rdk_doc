# 1.7.4 烧录系统


:::info 提示

- 如果您的存储设备已完成过系统烧录，可跳过此章节直接进行[添加 RDK 设备](../09_RDK_Studio/05_Device_management/01_hardware_resource.md)。
- 需使用第三方工具 XBurn 烧录系统。
:::


1. 点击下载烧录工具。
   
   ![工具下载界面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/s100/install_os_tools.png)
   
2. 点击下载系统镜像。
   
   ![镜像下载界面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/s100/os_image.png)
   
3. 点击 `了解更多` 进入产品界面。
   
   ![产品界面](http://rdk-doc.oss-cn-beijing.aliyuncs.com/doc/img/01_Quick_start/image/rdk_studio/s100/more.png)
