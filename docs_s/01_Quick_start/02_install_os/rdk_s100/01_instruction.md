# 1.2.1.1 烧录说明


:::warning 注意事项

- 禁止带电时拔插除 USB、HDMI 和网线之外的任何设备。
- 选用正规品牌的电源适配器，否则会出现供电异常，导致系统异常断电的问题
- 建议使用板载 POWER ON/OFF 按键实现主板上下电，并在适配器断电状态下对 DC 头进行插拔。

:::


## 烧录模式

RDK S100 开发套件可借助 PC 端工具 D-Navigation 完成 Ubuntu 系统的烧录工作，烧录过程支持 U-Boot 和 USB 两种模式。

### U-Boot 模式： 

- 此模式依赖 RDK S100 进入 U-Boot 的烧录模式（即 fastboot 模式）。
- 在日常的烧录场景中使用较为频繁，能满足大多数常规的系统烧录需求。


### USB 模式： 

- 该模式基于 DFU 协议，当 RDK S100 遇到无法进入 U-Boot 模式，或者系统损坏导致设备变砖等特殊情况时，使用此模式帮助恢复系统。
- 适用于空片烧录或者固件损坏无法进入 Uboot 情况。
