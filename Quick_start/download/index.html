注意事项​

1. 版本兼容性​

烧录前请确认硬件型号与镜像匹配。

  • RDK X3 派 1.0 版本需下载 1.x.x 目录镜像

  • RDK X3 Module 需搭配树莓派 CM4 载板使用

2. 烧录说明​

  • RDK Module 系列 首次使用建议通过 烧录工具 升级系统

  • 系统版本查询命令：cat /etc/version 或 rdkos_info

3. 开源资源​

  • 系统源码托管于 D-Robotics GitHub
